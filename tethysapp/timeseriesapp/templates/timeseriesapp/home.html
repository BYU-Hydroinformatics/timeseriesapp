Time Series Application Instructions

This Tethys application is for demo purposes in the different ways in which one can retrieve time series from different data sources. Some of these data sources includes, but are not limited to:

  • Application programming interface (API)

  • Csv / JSON files

  • Web Services Data

The following image summarizes the purpose of the application as a application to teach teh retrieval of times series from different sources to geometric features:



The TimeSeries app will show the same escenario for the different data sources of the time series. A map will be displayed with markers and Polygons, and time series will be added to such gemoetric features using the different data sources: API, Csv, JSON, Web Services. In other words, each gemotric feature will be assigned to a time serie from different data sources depending on the scenario. The API data source will retrieve time series for a REST and SOAP API, so the polygon and point features will be asigned time series from these data sources. Similarly, the data uploaded from a Csv and JSON formats will be assigned to a marker and polygon to display Time series. Finally, data from Geoservers can be assigned to a marker or polygon to display time series.

Each one of the tabs in the left has the different cases mentioned above. Each page is intuitive and easy to use, so just follow the isntructions when you get to each one of the pages that show how to assign time series to geometric features from different data sources.
