    API Instructions Modal

    This page tries to illustrate how one can create different features and assign to them time series throughout an API. The user will see a map, and a toolbar at the left hand site, so he/she can draw a polygon, customized marker, circle marker, circle, rectangle, and a polyline. Once the user choose and draw any of the above features. He/she can click on it to display the time series associated with them. Finally, the user can click on the white button to delete all the features and graphs associated with it.

    Note: the user can select only a feature at a time to display a time series in the bottom part of the screen
